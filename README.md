# Hidder

"Hidder" is a lightweight Fabric client mod that works with the Paper plugin "ModSeeker"  
It securely shares a player’s active mod list with the server, helping admins detect and manage client-side mods

### Features
- Collects and reports the player’s loaded mods to the server  
- Works seamlessly with the ModSeeker plugin  
- Lightweight and unobtrusive — no gameplay changes  

### Requirements
- Minecraft 1.21.8 (Fabric)  
- Server must have the ModSeeker plugin installed
